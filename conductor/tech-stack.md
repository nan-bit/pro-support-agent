# Technology Stack

## Core
- **Language:** Python 3.8+
- **Agent Framework:** [Google Agent Development Kit (ADK)](https://github.com/google/agent-development-kit)
- **AI Model:** Google Gemini Pro via `google-generativeai`

## Integrations
- **Platform:** [Shopify Admin API](https://shopify.dev/docs/api/admin-rest) (using `ShopifyAPI` Python library)
- **Environment Management:** `python-dotenv` for managing API keys and shop credentials.

## Infrastructure (Inferred)
- **Deployment/Execution:** Local or cloud-based Python environment suitable for running ADK-based web servers (Uvicorn).
